# 簡介
**此次作品是在資策會上完課程的成果發表，主要利用springboot框架製作飲料訂購平台，名為<<找茶>>，我負責的部分為購物車和訂單系統，運用ajax技術讓消費者在訂購飲料時不用重新整理網頁即可加入購物車內，還串接了綠界API來讓客戶進行結帳，並利用javamail api來讓客戶確認訂單，畫面呈現部分利用bootstrap5和datatable的套板來讓畫面更加美觀。**

# 使用技術
## 主要語言
- java
## 框架
- SpringBoot
## 資料庫
- SQL Server
## 資料庫存取技術
- Hibernate
## 前端
- jsp
- Bootstrap5
